#  To Do List in Java

This is a simple To Do List project developed in Java, focusing on practicing Object-Oriented Programming (OOP) concepts.

##  Features

- Add tasks
- List tasks
- Remove tasks
- Mark tasks as completed
- Mark tasks as pending

##  Concepts Practiced

- Object-Oriented Programming (OOP)
- Class organization and responsibility separation
- Use of ArrayList
- User input handling with Scanner
- Control structures (loops and conditionals)

##  Technologies

- Java

##  How It Works

The application runs in the terminal and allows the user to interact through a menu where they can manage their tasks.

##  Future Improvements

- Edit tasks
- Save tasks to a file
- Create a graphical interface
